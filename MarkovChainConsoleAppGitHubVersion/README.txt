				HOW TO USE

(All of the commands can typed into browser console)

Commands:
addSentence({sentence}): trains AI with given sentence(it can't be more than one sentence)
addText({text}): trains AI with given text(it can be more than one sentence)
predict({sentence}): if AI is trained you can call this function with an word or sentence you gived and it will list matching next words
